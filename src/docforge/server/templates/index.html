    • test_scout — counts test cases per framework, splits public_api into tested/untested.
    • api_scanner — HTTP routes (FastAPI/Flask/Express) + CLI commands (Click/Typer/argparse).
    • config_reader — env vars + .env files. Secret-redacts API_KEY/SECRET/TOKEN/… values.
    • reader — per-module RAG summary. Runs concurrently across modules.
    • architect — structured architecture: components, edges, external deps, runtime topology.
    • diagrammer — deterministic Mermaid + optional LLM beautify (validate-or-fallback).
    • writer — drafts README, ARCHITECTURE, API, TUTORIAL. Citation rule enforced in every prompt.
    • critic — verifies citations exist and ground; flags uncited claims and coverage gaps.
    • editor — revises drafts issue-by-issue. Loop bounded to 2 cycles.
